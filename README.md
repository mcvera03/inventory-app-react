Inventory App - Vite + React + Supabase

Quick start:
1. Copy .env.example to .env and fill VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY.
2. Push the SQL in schema_supabase.sql to Supabase SQL editor and run.
3. npm install
4. npm run dev

Netlify:
- Set env vars in Site settings (VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY)
- Build command: npm run build
- Publish directory: dist
