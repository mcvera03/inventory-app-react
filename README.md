# Inventory App (React + Supabase)

This is a ready-to-deploy React + Vite project connected to Supabase.
- Put your VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY in Netlify env vars.
- `npm install` then `npm run build` on Netlify will produce `dist`.

Structure: src/components, src/pages, src/lib/supabaseClient.js
